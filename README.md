
## Routes
### ToDo Routes:
- POST /todos/ 
  Description: Creates a newtodo. 
  Input Body: {title: 'title', description: 'desc' }

- GET /todos
  Description: gets all todos.

  Output: { the all todos } 

- GET /todos/:id
  Description: gets the specific todo with that id.

  Output: { the todo } 

- PUT /todos/:id
  Description: updates the todo with the given id
  Input: body with the todo in it
